# Math.c/h
These files contain the prototype and implementation for a function, 
_math_ that takes two integer arguments, _num1_ and _num2_ and a char 
argument, _Operator_ and returns and integer. _Operator_ can be a '+', '-', '*', '/', or '%'. 
The function will then add, subtract, multiply, divide, or modulo, 
_num1_ and _num2_ according to the _Operator_ passed into the function.
